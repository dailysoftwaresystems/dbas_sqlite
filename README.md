# DBAS.SQLite.Flutter
Flutter plugin that access SQLite for Android, iOS, macOS, Linux, Windows and Web

# Setup

### Web setup
For web you'll need to have [`dbas_sqlite.js`](https://github.com/dailysoftwaresystems/DBAS.SQLite.Flutter/blob/6c484892308ce50e0fef7ead8593a96a4a6334e7/native_libs/sqlite/web/dbas_sqlite.js)
inside your web/libs folder. It's strongly recommended to use the latest version of the file and a script to automatically copy it to the web/libs folder.
